# Shubham Tapadiya - Software Developer Portfolio
A clean, professional portfolio website showcasing 6+ years of software engineering experience at leading companies including E2Open and AWS. Built with React, Tailwind CSS, and Framer Motion.
## About
This portfolio demonstrates expertise in building scalable distributed systems, cloud architecture, DevOps, and performance optimization at industry-leading companies.
## Tech Stack
- React 18.2.0
- Tailwind CSS 3.2.7
- Framer Motion for animations
- React Intersection Observer
## Features
- Clean, professional design with no unnecessary animations
- Smooth scroll-triggered section animations
- Responsive on all devices (mobile, tablet, desktop)
- Easy to customize and update
- Contact information and resume download
- 4 real professional experience positions
## Getting Started
### Prerequisites
- Node.js 14.x or higher
- npm 6.x or higher
### Installation
```bash
cd portfolio
npm install --legacy-peer-deps
```
### Running Locally
```bash
npm start
```
Opens at http://localhost:3000
### Building for Production
```bash
npm run build
```
Generates optimized build in build/ folder.
## Customization
Edit src/data.js to modify your experiences and information.
## Deployment
Deploy to Netlify, Vercel, or GitHub Pages using standard React deployment methods.
